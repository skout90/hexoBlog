# introduce

`hexo framework`를 이용한 공부한 것을 기록하고 공유하는 블로그입니다.(2019-09-20 새버전 updated)

## setup

> npm i -g hexo-cli

> npm i

> cd themes/transquilpeak && npm run build

- 테마 수정시마다 아래 커맨드를 실행해야 적용됩니다.

> cd themes/transquilpeak && npm run build

## 블로그 포스팅 배포

> npx hexo deploy -g
